Verbatim Tests

The \verb macro prints its “argument” without any further processing. In HTML this is represented by the code element.

cat

Various characters can be used to delimit the contents of the verb command. Even { can be used!

The verbatim environment prints blocks of text without any further processing. In HTML this is represented by the pre element.

Here is text in a verbatim environment.

\begin{theorem}
$5$ is odd.
\end{theorem}
